# Required Images

The site now ships with real JPG photography sourced from Pexels for the current heroes, destination cards, regional contact cards, and program role cards.

SVG files remain in place as local fallbacks if a JPG is removed.

See `CREDITS.md` in this folder for the exact source pages and downloaded image URLs.

## Hero Backgrounds (1920×1080, landscape, dark enough for white text overlay)

| File | Subject |
|------|---------|
| `hero-home.jpg` | Scenic Canadian landscape (mountains, lake, city skyline) |
| `hero-program.jpg` | People connecting, handshake, or world map |
| `hero-contacts.jpg` | Canadian cityscape or travel scene |
| `hero-plan.jpg` | Traveller with map/backpack, or Canadian road/trail |
